# CoreShop Catalog Price Rules

## Default Conditions and Actions

CoreShop comes with a set of default action and condition implementations:

### Default Conditions


 - Categories
 - Products
 - Quantity
 - Timespan
 - Countries
 - Currencies
 - Customer Groups
 - Customers
 - Stores
 - Zones
 - Nested

### Default Actions

 - Discount Amount
 - Discount Percent
 - Price
